# Stack Overflow Survey Dashboard

## Project Overview
This project analyzes the Stack Overflow Developer Survey data and presents insights about current technology usage, future technology trends, and developer demographics.

The dashboard was created using **IBM Cognos Analytics** to visualize programming language preferences, database usage, platforms, web frameworks, and demographic information of survey respondents.

---

## Dataset
**Source:** Stack Overflow Developer Survey

Dataset used:
- `survey_data_updated.csv`

The dataset contains responses from developers worldwide, including:
- Programming languages
- Databases
- Platforms
- Web frameworks
- Country
- Age
- Education level

---

## Tools Used
- IBM Cognos Analytics
- CSV Dataset
- Data Visualization
- Dashboard Design

---

# Dashboard Sections

## 1. Current Technology Usage

This dashboard shows technologies developers currently use.

Visualizations:
- Top 10 Programming Languages Used
- Top 10 Databases Used
- Platforms Developers Have Worked With
- Top 10 Web Frameworks Used

---

## 2. Future Technology Trends

This dashboard highlights technologies developers want to learn or use in the future.

Visualizations:
- Top 10 Desired Programming Languages
- Top 10 Desired Databases
- Desired Platforms
- Top 10 Desired Web Frameworks

---

## 3. Demographics

This dashboard analyzes respondent characteristics.

Visualizations:
- Respondent Distribution by Age
- Respondent Count by Country
- Education Level Distribution
- Age Distribution by Education Level

---
## Key Insights
- Programming language popularity varies across developers based on current and future preferences.
- Developers show interest in emerging technologies and frameworks.
- Survey responses represent developers from multiple countries and education backgrounds.
